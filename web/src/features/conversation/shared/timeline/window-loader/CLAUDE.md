# 可见窗口加载

- `visible-round-candidate.ts`: 从未加载占位元素中按滚动方向和焦点距离选择下一候选。
- `window-loader-runtime.ts`: 维护会话代次、在途请求、完成集合、有限重试账本和滚动方向。
- `use-visible-window-loader.ts`: 只编排 RAF、退避定时器、滚动事件和异步加载阶段。

窗口加载与导航跳转保持不同策略：窗口加载选择视口候选并有限重试，跳转加载服务明确的最新目标；不得抽成丢失业务语义的通用队列。
加载结果使用 `loaded/missing/failed` 判别联合，结果必须先通过请求代次校验，再写入完成集合、重试账本或日志。
