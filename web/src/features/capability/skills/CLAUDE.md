# 技能市场

## 目录边界

- 根目录只保留页面编排、Header、搜索入口和导览定义。
- `controller/` 管理目录查询、外部搜索、来源配置、异步操作与反馈状态。
- `catalog/` 负责已安装技能目录和更新提示。
- `external/` 负责社区来源结果、来源管理和预览。
- `import/` 负责本地与 Git 导入表单。
- `detail/` 负责技能详情、Markdown 呈现和部署失败说明。

## 约定

- 子视图只接收自身消费的小接口，禁止传入完整市场控制器。
- 外部技能身份统一使用 `externalSkillKey`，导入冲突统一使用 `getExternalSkillImportState`。
- 异步请求的竞态由所属 controller 处理，视图不得自行维护请求版本或修正结果。
- 新增流程时先确定状态归属，再决定目录；不得把领域逻辑放回通用 `hooks/`。
