# 定时任务看板

- `scheduled-task-board.tsx` 只处理加载、错误、建议空态和四列看板装配。
- `scheduled-task-card.tsx` 负责单项必要信息与操作菜单，不请求 API，也不解释运行协议。
- `scheduled-task-board-model.ts` 是状态分列、排序、卡片文案和建议预设的单一真相源。
- 看板状态由任务事实自动投影，不提供拖拽或人工状态迁移；历史执行结果不伪装成独立工作流阶段。
- 看板占满页面头部以下的剩余高度；横向滚动归看板容器，纵向滚动归各列任务区，页面主体不再参与滚动。
- 卡片只显示任务名称、指令、计划、当前时间状态和失败摘要；执行会话、投递目标等高级字段留在编辑与历史视图。
